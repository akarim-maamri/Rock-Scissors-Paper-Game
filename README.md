the game "Rock, Paper, Scissors." This is a simple hand game often used to make a decision between two people. The gestures represent:

Rock (fist): The hand is clenched into a fist.
Paper (open hand): The hand is open and flat.
Scissors (peace sign): The index and middle fingers are extended, resembling a pair of scissors.
The rules of the game are:

Rock crushes Scissors
Scissors cut Paper
Paper covers Rock
Each player simultaneously forms one of the three shapes, and the winner is determined based on the rules above. If both players choose the same shape, the game is a tie and usually repeated.
